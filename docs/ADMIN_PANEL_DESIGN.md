# IdosoLink Admin Panel - UI/UX Design Document

## Design System

### Aesthetic Direction: **Minimal Premium SaaS**
- Clean, professional interface for platform management
- Generous whitespace for readability
- Subtle elevation and borders
- High contrast for data-heavy tables
- Clear visual hierarchy

### Color Palette

```css
/* Light Mode */
--admin-bg: #f8fafc;           /* Main background */
--admin-surface: #ffffff;       /* Cards, panels */

--admin-text: #0f172a;          /* Primary text */
--admin-text-secondary: #475569; /* Secondary text */
```

## Accessibility

- All tables: Keyboard navigable
- All modals: Focus trap + escape close
- All forms: Proper labels + error messages
- Color contrast: WCAG AA minimum
- Screen reader: ARIA labels on icons
